#### Как улучшить качество при VBR 

## Съёмка  

| Приём                   | Почему работает                             |
| ----------------------- | ------------------------------------------- |
| Статичный фон           | B/P-кадры почти пустые → биты на детали     |
| Штатив / стабилизатор   | Нет микродвижений камеры                    |
| Избегать мелких текстур | Листва, ткань, волосы — кошмар для кодека   |
| Размытый фон (боке)     | Меньше деталей вне фокуса = экономия        |
| Плавные движения        | Резкие = много различий между кадрами       |
| Чистый ISO              | Шум = случайные данные, кодек их "кодирует" |
| Меньше объектов в кадре | Проще сцена = выше качество каждого объекта |
## Монтаж  
  
| Приём                        | Почему работает                      |
| ---------------------------- | ------------------------------------ |
| Длинные планы                | Меньше склеек = меньше I-кадров      |
| Плавные переходы (dissolve)  | Резкий cut = новый I-кадр            |
| Избегать быстрых слайдшоу    | Каждый слайд = почти I-кадр          |
| Убрать зернистость (Denoise) | Шум — враг сжатия                    |
| Subtle color grading         | Агрессивный грейдинг создаёт бандинг |
| Статичные титры              | Движущийся текст жрёт биты           |
## Экспорт  
  
| Приём            | Почему работает                    |
| ---------------- | ---------------------------------- |
| 24 fps вместо 30 | Меньше кадров = больше бит на кадр |
| Main10 профиль   | Плавные градиенты без бандинга     |
