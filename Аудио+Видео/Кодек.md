это алгоритм кодирования и декодирования видео

Видеокодеки - используются для видео, бывают:

- Монтажные кодеки (быстро записывают быстро воспроизводят)
-много весят
DNxHD, Apple ProRes

- Просматровые (медленно записывают быстро воспроизводят)
+высокая скорость декодирования
+высокая компрессия
+мало весят
-низкая скорость кодирования

| VP9             | кодек от Google                            | старый            |                                                                |                                                                              |
| AV1             |  имба                                      |                   |                                                                |                                                                              |

 +высокая скорость кодирования видео в камере
 -низкая скорость декодирования
 -низкая компрессия
