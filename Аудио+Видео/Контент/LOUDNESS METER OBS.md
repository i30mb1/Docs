# Настройка аудио для стрима / видео

**Стек:** GoXLR Mini → OBS → Youlean Loudness Meter  
**Платформы:** YouTube · Twitch · TikTok

---

## Цепочка обработки

```
Микрофон → [GoXLR: Gate → EQ → Compressor] → [OBS: Limiter] → Стрим/Запись
```

Все фильтры на голос — в GoXLR. В OBS только Limiter как страховка.

---

## Шаг 1 — Диагностика чистого сигнала

Сбросить всё в GoXLR (Gate = 0, EQ = 0, Compressor = 0). В Youlean нажать Reset, говорить 15–20 сек.

| Параметр | Цель | Мои значения |
|---|---|---|
| Short Term LUFS | –16…–18 LUFS | ______ |
| True Peak Max | ≤ –1 dBTP | ______ |
| Метр OBS | Жёлтая зона | ______ |

> Если True Peak уже близко к –1 dBTP — Gain настроен правильно, не трогать.

---

## Шаг 2 — Gate (GoXLR)

| Параметр    | Значение   | Зачем                                       |
| ----------- | ---------- | ------------------------------------------- |
| Threshold   | **–30 dB** | Открывается на голосе, закрывается в паузах |
| Attenuation | **100%**   | Полная тишина в паузах                      |
| Attack      | **10 ms**  | Нет обрезания начала слов                   |
| Release     | **200 ms** | Плавное закрытие                            |

Если срезает начало слов → уменьши Threshold
Если слышна клавиатура в паузах → подними Threshold до –25 dB

**Мои значения:** Threshold ______ · Attack ______ · Release ______

---

## Шаг 3 — EQ (GoXLR)

| Полоса  | Значение | Зачем                       |
| ------- | -------- | --------------------------- |
| 90 Hz   | **–9**   | Убрать гул и вибрацию стола |
| 160 Hz  | 0        | Тело голоса — не трогать    |
| 480 Hz  | 0        | —                           |
| 1.5 kHz | 0        | —                           |
| 4.5 kHz | **+2**   | Разборчивость речи          |
| 7.8 kHz | **+1**   | Лёгкое присутствие          |

**Мои значения:** 90Hz ______ · 4.5kHz ______ · 7.8kHz ______

---

## Шаг 4 — Compressor (GoXLR)

| Параметр | Значение | Зачем |
|---|---|---|
| Threshold | **–18 dB** | На 2–4 dB ниже уровня нормальной речи |
| Ratio | **4:1** | Мягкое сжатие, голос звучит живо |
| Attack | **10 ms** | Сохраняет твёрдость согласных |
| Release | **100 ms** | Нет пампинга между словами |
| Make-up Gain | **+5 dB** | Компенсация — поднять до –16 LUFS Short Term |

Проверка: Gain Reduction при речи = –3…–6 dB  
Если GR = 0 → Threshold слишком высокий  
Если GR постоянно –10 дБ → Threshold слишком низкий

**Мои значения:** Threshold ______ · Make-up Gain ______

---

## Шаг 5 — Limiter (OBS)

Единственный фильтр на микрофон в OBS. Ставится последним.

| Параметр | Значение |
|---|---|
| Threshold | **–1 dB** |
| Release | **60 ms** |

---

### Компрессор на игровой звук (OBS)

Игра динамичнее голоса — взрывы резкие, нужно агрессивнее давить пики.

| Параметр | Значение | Отличие от голоса |
|---|---|---|
| Ratio | **8:1–10:1** | Выше — пики резче |
| Threshold | **–30…–35 dB** | Ниже — игра динамичнее |
| Attack | **1–5 ms** | Быстрее — поймать взрыв |
| Release | **150 ms** | Чуть дольше — игра хаотичнее |
