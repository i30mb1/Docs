## Стандарты громкости по платформам (2025 год)

Ориентируйтесь на эти значения при финальном сведении или настройке мастер-шины в OBS.

| **Платформа**      | **Целевой LUFS (Integrated)** | **True Peak (dBTP)** | **Особенности нормализации**                            |
| ------------------ | ----------------------------- | -------------------- | ------------------------------------------------------- |
| **YouTube**        | -14 LUFS                      | -1.0 dBTP            | Только понижает громкое видео                           |
| **Twitch**         | -16...-14 LUFS                | -1.0 dBTP            | Нет жесткой автоматики; риск «клиппинга»                |
| **TikTok / Reels** | -12...-10 LUFS                | -1.5...-2.0 dBTP     | Ценится высокая плотность звука для мобильных динамиков |

> YouTube не поднимает тихий контент. Загружай не тише –14 LUFS, иначе ролик будет тише конкурентов.
## Единая таблица производства (Уровни и Динамика)

В этой таблице указаны целевые значения для настройки каждого канала в вашем микшере (OBS, Voicemeeter и др.).

| **Элемент / Тип контента** | **Громкость (LUFS Int.)** | **Диапазон (LRA)** | **Роль и результат в миксе**                                |
| -------------------------- | ------------------------- | ------------------ | ----------------------------------------------------------- |
| **Ведущий голос (Mic)**    | **-16...-14**             | **3...5 LU**       | Стабильный «якорь»; голос не пропадает и не кричит          |
| **Discord / Гости**        | **-19...-17**             | **3...6 LU**       | Эффект присутствия; голос четкий, но чуть тише автора       |
| **Игра (Gameplay)**        | **-30...-26**             | **5...10 LU**      | Погружение; взрывы ощутимы, но не перебивают речь           |
| **Фоновая музыка**         | **-45...-35**             | **2...4 LU**       | Едва заметный фон; не отвлекает от основной сути            |
| **TikTok / Shorts**        | **-12...-10**             | **2...5 LU**       | Максимальная плотность для мобильных динамиков              |
| **Весь микс (Master)**     | **-14**                   | **6...9 LU**       | Итоговый баланс для YouTube/Twitch; комфортно слушать долго |
Важный нюанс: Каждый раз, когда ты меняешь Threshold или Ratio, показатель Integrated LUFS будет меняться. Поэтому сначала всегда настраивай «плотность» (LRA), а уже в самом конце — финальную громкость ручкой Gain.
