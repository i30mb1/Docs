это файл хранящий закодированное содержимое

Контейнер с [[Расширение|расширением]] .wav будет хранить в себе содержимое в формате .wav
Контейнер c [[Расширение|расширением]] .jpeg будет хранить в себе содержимое в формате .jpeg

С видео контейнерами все сложнее, нельзя сказать каким [[Кодек|кодеком]] закодировано видео

| **Контейнер** | **Поддерживаемые видеокодеки** | **Поддерживаемые аудиокодеки** |
| ------------- | ------------------------------ | ------------------------------ |
| **MP4**       | H.264, H.265, MPEG-4           | AAC, MP3, FLAC                 |
| **AVI**       | DivX, Xvid, MPEG-2, H.264      | MP3, AAC, PCM                  |
| **MOV**       | H.264, H.265, ProRes, DNxHD    | AAC, PCM, ALAC                 |
| **WebM**      | VP8, VP9, AV1                  | Opus, Vorbis                   |
